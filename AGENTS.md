# AI Agent Instructions

## Project overview
This repository contains Python practice exercises for file handling and text processing. The main source files live under `file_reading_practice/` and each file implements a single problem statement such as palindrome detection, longest palindrome search, vowel coverage, and sonnet analysis.

## Key files
- `file_reading_practice/letters_never_back_to_back.py`
- `file_reading_practice/longest_palindrome.py`
- `file_reading_practice/palindrome_words.py`
- `file_reading_practice/sonnet_minus_sowpods.py`
- `file_reading_practice/sonnet_words.txt`
- `file_reading_practice/sowpods.txt`
- `file_reading_practice/words_with_all_vowels.py`

## Agent guidance
- Preserve the existing problem descriptions and function names in the Python files.
- Use the provided text files in `file_reading_practice/` when fixing or extending logic.
- Keep solutions simple and idiomatic Python; no external packages are required.
- If adding execution examples, use `python file_reading_practice/<script>.py` from the repo root.

## Useful context
- The repository is a learning assignment rather than a production application.
- There are no explicit tests or setup scripts in the repo.
- Most code is self-contained in single-file Python scripts.

## When editing
- Do not introduce complex frameworks or unnecessary dependencies.
- Maintain consistency with the existing naming and file-handling style.
- If a script reads from `sowpods.txt` or `sonnet_words.txt`, ensure the path is relative to `file_reading_practice/` when run from the repo root.
